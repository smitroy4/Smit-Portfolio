Rental Booking Backend System

Overview

A scalable backend system designed to manage users, property listings, and bookings. Built using Spring Boot and PostgreSQL with a focus on real-world backend challenges such as authentication and booking conflicts.

Tech Stack

Java Spring Boot Spring Security JPA PostgreSQL

Key Features

  • JWT-based authentication and role-based access
  • Booking conflict handling system
  • Pagination and filtering APIs
  • Clean layered architecture (Controller-Service-Repository)

System Design

The system follows a layered architecture where controllers handle HTTP requests, services manage business logic, and repositories interact with the database. Authentication is handled using JWT tokens to maintain stateless sessions.
